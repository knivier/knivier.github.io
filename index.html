Activerse - Java

My biggest project to date as of right now, Activerse is a Java-based game engine meant to help people create games by simplifying the GUI experience. I was inspired by Greenfoot but annoyed from the lack of customization and descriptions regarding how Greenfoot initially worked, and I wanted a more customizable version of which I could modify the source code. From a library to engine, this is my biggest project to date. For more information on how to use this project, you can check out the Wiki which has all the quick start guides you'll need.

Troll Game - Java

Troll Game is a simple text-based game that uses logic and lots of randomization to make each play seem different from the previous. I created this game for my AP-CSA midterm, and I've tried to keep it alive to this day! While updates are on the slower slide, it's still a well polished game that can be entertaining (depending on your genre type).

Pal - Python

Pal is a simple Discord bot that's used as a template to create other things. I originally created it in 2020 during COVIDs reign with the help of a friend. However, since then, it's become a much more customizable Discord bot that helped me see how Python's syntax and usability was. The project is updated on a much slower pace because Python isn't my primary (nor favourite) language.
