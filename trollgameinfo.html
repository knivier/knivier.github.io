Hi, I'm Knivier! I'm the solo developer behind Troll Game, created in January 2024. This project has been sunsetted as of 7/18/2024 and will only receive updates for bugs and/or security purposes.
Troll game was never meant to be the project I always worked on, it was simply a hobby to kill time. Text-based games are unpopular and rather difficult to show off as an achievement, and the code is not at my level anymore. I don't have the time to update the project with new features, because my priorities are elsewhere. Feel free to add onto what I could not, in this still awesome text-based game.

Game Information

The Game

Troll Game puts you in charge of leading the trolls, engaged in an eternal battle against Archie the Dragon. Choose between Ranked Mode for quick gameplay focused on earning money, or Adventure Mode (in progress) for a deeper dive into the game mechanics.

How to Get Playing?

To start playing, download the JAR file and ensure you have Java JDK 17 or higher installed. Optionally, use the run.bat file to launch the game without a terminal. Visit GitHub for the latest version.

Can I Contribute?

You can mod the game using IntelliJ IDEA or BlueJ. For modding details, visit the Developers Insider. Report bugs and suggest ideas via Discord.
